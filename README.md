# conan.install

**Conan.io Plugin for Windows Binary packes** 

## Prerequisites
  [maiken](https://github.com/Dekken/maiken)


## Example use case

https://github.com/mkn/mkn.ram/blob/master/mkn.yaml#L32


## Building
  
  Windows cl:

    mkn clean build -tSa -EHsc -d


## Testing
  
  Windows cl:

    mkn clean build -tSa -EHsc -dp test run
